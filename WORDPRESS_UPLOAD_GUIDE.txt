METHOD 1: DOWNLOAD & UPLOAD (EASIEST) ‚≠ê
================================================================================

STEP 1: Download HTML files to your computer
--------------------------------------------
From your local computer, run:

scp -r root@YOUR_VPS_IP:/root/synexs/wordpress_html ./synexs_docs

(Replace YOUR_VPS_IP with your actual VPS IP address)

This downloads all HTML files to your computer in ./synexs_docs/


STEP 2: Upload to WordPress
----------------------------
For EACH HTML file:

1. Go to: https://synexs.net/wp-admin/post-new.php?post_type=page

2. Add page title (e.g., "Synexs Documentation")

3. Click the (+) button to add block

4. Search for "Custom HTML" block

5. Open the HTML file in a text editor (Notepad, VSCode, etc.)

6. Copy ENTIRE content (Ctrl+A, Ctrl+C)

7. Paste into WordPress Custom HTML block

8. Click "Preview" to check formatting

9. Set page URL slug (e.g., "documentation")

10. Click "Publish"

METHOD 2: WORDPRESS REST API (AUTOMATED) ‚ö°
================================================================================

STEP 1: Generate WordPress App Password
----------------------------------------
1. Go to: https://synexs.net/wp-admin/profile.php
2. Scroll to "Application Passwords"
3. Name: "Synexs Upload"
4. Click "Add New Application Password"
5. COPY the password (format: xxxx xxxx xxxx xxxx)


STEP 2: Configure the script
-----------------------------
Edit wordpress_upload.py:

nano wordpress_upload.py

Change:
WP_USERNAME = "your_username"
WP_APP_PASSWORD = "xxxx xxxx xxxx xxxx"  # Paste your app password


STEP 3: Install dependency
---------------------------
pip3 install requests markdown


STEP 4: Run the uploader
-------------------------
python3 wordpress_upload.py

This will:
- Convert all docs to HTML
- Upload to WordPress as DRAFT pages
- Show you the URLs
- You review drafts and publish manually
