## üéØ Core Implementation Files

### **binary_protocol.py**
- **Path**: `/root/synexs/binary_protocol.py`
- **Size**: 14KB
- **Purpose**: Binary encode/decode, hybrid protocol support
- **Functions**:
  - `encode_binary(actions)` - Encode to raw binary
  - `decode_binary(binary)` - Decode from binary
  - `encode_base64(actions)` - JSON-safe Base64
  - `decode_base64(b64_string)` - Decode Base64
  - `HybridProtocol()` - V1/V2/V3 unified interface
  - `generate_binary_training_data(n)` - Generate samples
- **Test**: `python3 binary_protocol.py`

### **vocab_v3_binary.json**
- **Path**: `/root/synexs/vocab_v3_binary.json`
- **Size**: 579 bytes
- **Purpose**: 32-action binary vocabulary
- **Format**: `{"ACTION": 0x00, ...}` (5-bit codes)

### **training_binary_v3.jsonl**
- **Path**: `/root/synexs/training_binary_v3.jsonl`
- **Size**: 266KB
- **Purpose**: 1000 training samples in binary format
- **Format**: JSONL with Base64-encoded sequences
- **Fields**: instruction, input, output, actions, protocol, format

## üîÑ Hybrid Support Files

### **cells/cell_001_hybrid.py**
- **Path**: `/root/synexs/cells/cell_001_hybrid.py`
- **Size**: 6KB
- **Purpose**: Multi-protocol sequence generator
- **Modes**:
  - `SYNEXS_PROTOCOL=v1` - Greek words
  - `SYNEXS_PROTOCOL=v2` - Symbols
  - `SYNEXS_PROTOCOL=v3` - Binary
  - `SYNEXS_PROTOCOL=v3-hybrid` - All three (default)
- **Output**: `datasets/generated/generated_hybrid_*.json`
- **Test**: `python3 cells/cell_001_hybrid.py`

### **protocol_demo.py**
- **Path**: `/root/synexs/protocol_demo.py`
- **Size**: 5.3KB
- **Purpose**: Visual comparison of V1/V2/V3 protocols
- **Scenarios**: 5 real-world message examples
- **Metrics**: Size, speed, reduction percentage
- **Test**: `python3 protocol_demo.py`
